# 📚 Book AI RAG App

## Description
This project is a full-stack AI application that:
- Scrapes books using Selenium
- Stores data in Django backend
- Generates summaries using AI
- Answers questions using RAG pipeline

## Features
- Book listing
- AI Summary
- Question Answering (RAG)

## Setup

Backend:
pip install -r requirements.txt
python manage.py runserver

Frontend:
npm install
npm start

Q: What is Sapiens about?
